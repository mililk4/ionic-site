Contribute To Ionic

Help build awesomness! If you see an area for improvement, or have an idea for a new feature then we'd love to have your help!

  1. Visit issues for ideas or any discussions which may be taking place.
  2. Fork the Github repo.
  3. Create a new branch from master: git checkout -b my_awesomeness
  4. Develop your feature.
  5. Follow our coding conventions.
  6. Submit a pull request!

Submit An Issue On Github

If you catch a typo, or see a way to improve any piece of code please let us know. There's no such thing as too small of an improvement.

Conventions

We try to follow best practices as much as possible.

Prior to any CSS (SASS) being committed we prefer the code is ran through CSS Comb first, which is "The Greatest tool for sorting CSS properties in specific order".

Additionally, all JavaScript must pass JSHint, which is "a tool that helps to detect errors and potential problems in your JavaScript code."
