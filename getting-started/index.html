Install Ionic

First, install Node.js. Then, install the latest Cordova and Ionic command-line tools.

{% highlight bash %} $ npm install -g cordova $ npm install -g ionic {% endhighlight %}

Start a project

Create a project with our pre-built seed project. You can also clone the seed project itself, which gives you everything you need to build Ionic apps with Cordova and AngularJS.

{% highlight bash %} $ ionic start myApp {% endhighlight %}

Run it

Ionic apps are based on Cordova, so we can use the Cordova utilities to build, test, and deploy our apps, but we provide simple ways to do the same with the ionic utility (substitute ios for android to build for Android):

{% highlight bash %} $ cd myApp $ ionic platform ios $ ionic emulate ios $ ionic run ios {% endhighlight %}

That's it! To learn more, visit the Ionic Documentation or ask a question on the Forum.
