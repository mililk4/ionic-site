Install Ionic

Install the latest Ionic command-line tool. You can also clone it on GitHub to get the bleeding edge release.

{% highlight bash %} $ npm install -g ionic {% endhighlight %}

Start a project

Create a project with our pre-built seed project. You can also clone the seed project itself, which gives you everything you need to build Ionic apps with Cordova and AngularJS.

{% highlight bash %} $ ionic start myApp {% endhighlight %}

Run it

Ionic apps are based on Cordova, so we can use the Cordova utilities to build, test, and deploy our apps:

{% highlight bash %} $ cd myApp $ cordova platform add ios $ cordova emulate ios {% endhighlight %}
