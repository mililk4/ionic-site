About Ionic

Ionic is a 100% open source HTML5 mobile framework, with a permissive MIT license that can be used in any kind of app. Ionic was created by Drifty, a developer tools company behind such projects as Codiqa and Jetstrap.

Why we built it

We built Ionic to take HTML5 on mobile into the future. We wanted a mobile framework that looked great, and was powerful enough to build the amazing apps the best developers were building.

We knew that HTML5's time had come, and we wanted to build the framework we wish existed. We knew a great HTML5 framework had to look amazing, be incredibly fast, and be fun to develop in.

Read more about why we built Ionic on our blog

License FAQs

Ionic is an open source framework released under a permisive MIT license. This means you can use Ionic in your own personal or commercial projects for free.
